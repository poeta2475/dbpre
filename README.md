# DEOSOLUCIONES S.A.S — Presentación web

Versión web de la presentación, lista para **GitHub Pages**.

- `index.html` — una sola página con las 18 diapositivas y transiciones en CSS.
- `images/` — recursos extraídos de la presentación original.

## Ver en GitHub Pages

1. Settings → Pages.
2. En **Source** elige la rama `lan` y carpeta `/ (root)`.
3. Abre la URL que GitHub genere.

## Controles

- Flechas ← → o barra espaciadora para navegar.
- Clic en los puntos inferiores para saltar.
- Botón ⛶ para pantalla completa. Swipe en móvil.
